# CardEcho Desktop 系统使用说明书

## 📚 产品概述

CardEcho Desktop 是一款基于AI技术的智能语言学习应用，通过创新的"卡片式学习"模式，帮助用户高效掌握英语语言技能。系统结合了先进的AI分析、语音技术和游戏化学习体验，为用户提供个性化的语言学习解决方案。

### 🎯 核心理念
- **智能拆解**：将复杂文本拆解为易于学习的小单元
- **重复记忆**：通过科学的复读机制强化记忆效果
- **游戏化激励**：结合挑战模式激发学习兴趣
- **数据驱动**：基于学习数据提供个性化建议

## 🚀 快速开始

### 1. 首次使用
1. 启动应用后，系统会引导您完成用户注册/登录
2. 设置您的学习目标（默认每日20张卡片）
3. 选择您偏好的AI引擎和语音服务
4. 开始您的语言学习之旅

## 📖 核心功能模块

### 1. 🏠 资源库 (Library)
**功能描述**：管理您创建和订阅的所有学习卡包

**主要操作**：
- **浏览卡包**：查看所有可用的学习资源
- **创建新卡包**：通过AI分析文本创建个性化学习内容
- **订阅/取消订阅**：从商城获取优质学习资源
- **删除卡包**：管理个人学习资源

**使用技巧**：
- 使用标签系统对卡包进行分类管理
- 定期清理已完成学习的卡包
- 利用"已上架商城"标识管理发布状态

### 2. 🎵 训练模式 (Training Mode)
**功能描述**：核心学习模块，通过自动播放和复读强化记忆

**学习流程**：
1. 选择要学习的卡包
2. 系统自动播放每张卡片3-5次
3. 支持语速调节（0.75x-1.5x）
4. 自动记录学习进度

**特色功能**：
- **智能语音播放**：多语音引擎支持（Google TTS、豆包语音）
- **语速调节**：根据个人听力水平调整播放速度
- **自动进度跟踪**：实时记录学习状态
- **批量语音生成**：为整个卡包一次性生成语音

### 3. 🎮 挑战模式 (Challenge Mode)
**功能描述**：游戏化学习体验，提升学习趣味性和挑战性

**挑战类型**：
- **听力训练**：根据听到的内容选择正确答案
- **拼写输入**：听写并输入听到的句子
- **快速反应**：在规定时间内完成挑战

**游戏规则**：
- 每轮挑战包含10个题目
- 根据准确率和速度计算得分
- 支持重听和跳过功能
- 提供实时反馈和鼓励

### 4. ✨ 内容创建器 (Content Creator)
**功能描述**：AI辅助的内容创作工具

**创建流程**：
1. **输入文本**：粘贴或输入英文文本
2. **AI分析**：系统自动拆解文本为学习卡片
3. **预览确认**：查看AI生成的卡片内容
4. **保存发布**：保存到个人库或发布到商城

**AI分析能力**：
- **词汇解析**：提供音标、词性、中文释义
- **语法分析**：识别句子结构和语法要点
- **语境说明**：解释使用场景和背景
- **学习建议**：提供个性化的学习指导

### 5. 🛒 资源商城 (Store)
**功能描述**：学习资源分享和获取平台

**商城功能**：
- **浏览资源**：查看所有公开的学习卡包
- **搜索筛选**：按标签、难度、热度等条件筛选
- **订阅资源**：一键订阅优质学习内容
- **发布管理**：管理个人发布的卡包

**发布规则**：
- 所有用户都可以发布自己的卡包
- 发布前需确保内容质量
- 支持卡包更新和删除
- 提供发布状态跟踪

### 6. 📊 学习统计 (Statistics)
**功能描述**：全面跟踪和展示学习进度

**统计指标**：
- **今日完成度**：显示当日学习进度百分比
- **累计训练卡片**：历史总学习卡片数量
- **预计训练时长**：基于学习时间的估算
- **连续达标天数**：连续完成学习目标的天数

**数据可视化**：
- 环形进度条展示当日完成情况
- 卡片式数据展示各维度统计
- AI学习洞察提供个性化建议
- 趋势分析帮助调整学习策略

## ⚙️ 系统设置

### 1. 语言设置
- **界面语言**：支持中文和英文界面切换
- **学习语言**：目前专注于英语学习

### 2. AI引擎配置
- **服务商选择**：Google Gemini / DeepSeek
- **模型选择**：Flash（快速）/ Pro（专业）/ Reasoner（推理）
- **语音提供商**：Google TTS / 豆包语音

### 3. 学习偏好
- **播放速度**：全局语速设置（0.75x-1.5x）
- **自动播放**：是否启用自动切换到下一张卡片
- **每日目标**：设置每日学习卡片数量
- **提醒设置**：学习提醒和进度通知
